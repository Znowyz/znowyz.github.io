Sobre este proyecto

Un proyecto de clase venido a más, el Sesgo Emético habla sobre los discursos de odio y aprender a lidiar con ellos, a identificarlos, y como se esconden bajo bulos, preconcepciones, y números vacíos.

Proceso creativo

4 páginas a color concebidas a partir de una noticia de un periódico local de Alcalá, que dio el pistoletazo de salida a un tebeo corto.
